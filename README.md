# Calculadora

Proyecto simple de calculadora web hecho con HTML, CSS y JavaScript.

## Descripción

Esta calculadora permite realizar operaciones básicas con una interfaz moderna y responsive.

### Funcionalidades

- Suma, resta, multiplicación y división
- Ingreso de números y punto decimal
- Botón de limpiar (`C`)
- Manejo de errores como división por cero

## Archivos principales

- `calculadora.html` - la interfaz de usuario de la calculadora
- `calculadora.css` - estilos visuales de la calculadora
- `calculador.js` - lógica de cálculo y control de eventos

## Cómo usar

1. Abre el archivo `calculadora.html` en tu navegador.
2. Haz clic en los botones para ingresar números y operadores.
3. Presiona `=` para ver el resultado.
4. Presiona `C` para reiniciar la pantalla.

## Notas

- La calculadora utiliza `eval()` para evaluar la expresión ingresada.
- Si se produce una operación inválida o una división por cero, se muestra `Error`.
